Registration

If you have registered for the Summer Institute, you do not need to register separately for BTS 2025.

If you haven't registered for the Summer Institute, you can do one of the following:

  • As a UBC student: You can register for the Summer Institute for free via this link
  • As a Non-UBC student: You can register for the Summer Institute for CAD$50 via this link
  • As a Speaker: You can register for free by sending an email to chen.feng@ubc.ca
  • Otherwise: You can register for BTS 2025 for CAD$200 via this link
